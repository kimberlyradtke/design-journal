Grid

What is it?

The grid is an effective way to line up all elements on a page in relation to one another. A grid allows a designer to create space with equal measurements. It aids in organization in layout. Working with a grid also streamlines web design to some degree. Web sites are built by many different people. If designers work with a common grid practice, their structure foundation may be relatively similar or have commonality.

Why I chose it?

I chose The Work Cycle to showcase effective use of the grid system. This site arranges many graphics that are all the same size. All of the graphics are perfectly aligned to one another with precise spacing. The result is a clean and organized presentation of information which could easily become chaos without use of a grid.
