Color and Value

What is it?

Color and value are an important aspect to web design. The colors chosen can set the mood, feel, and purpose of the website. The colors would change if the designers wanted a fun site versus a serious site. Color can also be used to create emphasis or highlight aspects of the page. Color is important because it catches the eye.common grid practice, their structure foundation may be relatively similar or have commonality.

Why I chose it?

I chose Grind Spaces to showcase effective use of color and value. This site is a great example of how color can be used to create emphasis. The initial photo used of the work space is a greyscale image. The words “work liquid” is written in red which says what the company is all about. Red is consistently used as a rollover element for the navigation links and to show the additional navigation menu on the bottom of the page.
